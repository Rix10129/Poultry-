Step 2 — Install Docker Desktop
1
Download Docker Desktop

Go to docker.com/products/docker-desktop → click Download for Windows

2
Run the installer

Double-click the downloaded file (Docker Desktop Installer.exe) and follow the on-screen steps. Click OK on all prompts. Your PC may restart.

3
Start Docker Desktop

After restart, open Docker Desktop from the Start Menu. Wait until the whale icon in the taskbar stops animating — this means Docker is ready.

  • The icon looks like a small whale 🐳 in the bottom-right taskbar
  • Wait until it shows "Docker Desktop is running"
⚠ Important

Docker Desktop must be open and running every time you want to use the Poultry Vet System. You do not need to do anything inside Docker Desktop — just keep it open in the background.

Step 3 — Copy the Software to the PC
1
Get the software folder

Your system provider will give you a USB drive or a download link. Copy the entire Poultry- folder to an easy-to-find location on the PC, for example:

C:\PoultryVet\
2
The folder should contain these files
  • docker\start.bat — starts the system
  • docker\stop.bat — stops the system
  • docker\backup.bat — creates a backup
  • docker\README.txt — quick reference
  • Dockerfile — (do not delete)
  • docker-entrypoint.sh — (do not delete)
  • docker\docker-compose.yml — (do not delete)
Step 4 — First-Time Start (One Time Only)
⏱ The first start takes 5–15 minutes

Docker needs to download and build the system the first time. This only happens once. Every start after this takes less than 1 minute.

1
Make sure Docker Desktop is running

Check the taskbar — you should see the whale icon 🐳 and it should not be animating.

2
Double-click start.bat

Go into your Poultry-\docker\ folder and double-click start.bat

A black window will open. You will see messages like:

================================================ Poultry Vet Distribution System ================================================ Starting services... Still starting... (15s) Still starting... (30s) ...
3
Wait for the browser to open automatically

When ready, your browser opens at http://localhost:3000 and the window shows:

================================================ Ready! Opening in your browser... http://localhost:3000 ================================================
4
Create your account

Go to http://localhost:3000/register and fill in:

  • Company Name (your business name)
  • Your Name
  • Email address
  • Password (minimum 8 characters)

After registering, your system provider will approve your account. You will receive an email confirmation, then you can log in.

✅ Your data is always safe

Your data is stored in a protected database on your PC. It is never deleted when you stop or restart the system — even if the PC restarts or loses power.

Backups — Very Important
🚨 Take a backup at least once a week

If your PC is stolen, breaks, or gets a virus, you could lose all your business data. Backups are your only protection. It takes less than 1 minute.

1
Run the backup

While the system is running, double-click docker\backup.bat

A backup file is saved in docker\backups\ with today's date in the name:

docker\backups\poultry_vet_2025-01-15_09-30.sql
2
Copy the backup to a USB drive

After every backup, copy the docker\backups\ folder to a USB drive or external hard disk. Keep the USB drive in a safe place — ideally away from the PC.

3
Recommended backup schedule
  • Daily — run backup.bat at end of day (if you entered data that day)
  • Weekly minimum — copy backups folder to USB drive
  • Monthly — copy to a second USB drive and store offsite
How to Restore a Backup

If you need to restore your data (e.g. after getting a new PC), contact your system provider. Provide them with the .sql backup file from your USB drive.

If you are restoring yourself:

1
Start the system first

Make sure start.bat has been run and the system is running.

2
Open Command Prompt in the docker folder

Hold Shift + right-click inside the docker\ folder → click Open PowerShell window here

3
Run the restore command
docker compose exec -T db psql -U poultry -d poultry_vet < backups\poultry_vet_2025-01-15_09-30.sql

Replace the filename with your actual backup file name.

Troubleshooting
ProblemSolution
"Docker Desktop is not running" Open Docker Desktop from the Start Menu. Wait for the whale icon to stop animating, then run start.bat again.
Browser shows error or blank page Wait 2 more minutes and refresh. First start after a PC restart is slow.
Site won't open at all Run stop.bat, wait 30 seconds, then run start.bat again.
Forgot password Go to http://localhost:3000/forgot-password and enter your email.
System is very slow Close other programs (Chrome tabs, etc.). Make sure your PC has at least 4 GB of RAM free.
backup.bat says "failed" Make sure the system is running first (run start.bat). Then try the backup again.
PC was restarted, system won't start Open Docker Desktop first, wait for it to fully load, then run start.bat.
Windows asks "Allow this app?" Click Yes / Allow. This is expected for start.bat and backup.bat.
Adding More Staff Users

You can add multiple staff members (salesmen, accountants, etc.) with different access levels.

1
Log in as Owner/Admin

Go to http://localhost:3000 and log in with your account.

2
Go to Users

Click Users in the sidebar → click Add User

3
Fill in staff details

Enter their name, email, and role. They will use these to log in on the same PC at http://localhost:3000.

Receiving Software Updates
ℹ When an update is available

Your system provider will contact you when an update is ready. Updates include new features, bug fixes, and security improvements.

1
Take a backup first

Always run backup.bat before updating and copy it to a USB drive.

2
Stop the current system

Run stop.bat

3
Replace the software folder

Copy the new Poultry- folder provided by your system provider over the old one. Do not delete the docker\backups\ folder — move it to the new folder.

4
Start again

Run start.bat — it will rebuild with the new version automatically.
